# HYPNOCHIP

[![pages-build-deployment](https://github.com/tjscooper/stuff/actions/workflows/pages/pages-build-deployment/badge.svg?branch=main)](https://github.com/tjscooper/stuff/actions/workflows/pages/pages-build-deployment)

Personal site for Tim Cooper — art, game dev, 3D printing, CNC, laser engraving, and daily habit tracking. Atari-inspired sci-fi fantasy aesthetic with a pixel art light theme.

Live at **[tjscooper.github.io/stuff](https://tjscooper.github.io/stuff)**

## Features

- **// CHRONICLES** — Blog/portfolio with category filters (Art, Game Dev, 3D Printing, CNC, Laser Engraving). Markdown editor with clipboard paste-to-upload for screenshots.
- **// MISSION LOG** — Private daily habit tracker with three groups: Chore, Tim, Work. Timeboxing, carried-over incomplete tasks with age indicators, and inspiration idea chips per column.
- **// STATS** — Flyout drawer with completion rates, streaks, time logged, and per-group breakdowns.
- **Daily asteroid** — Background generated each day via cellular automata seeded by the date, as a nod to Spacecavator.

## Stack

- Vanilla JS / HTML / CSS — no framework
- [Supabase](https://supabase.com) — Postgres DB, Auth, and Storage
- [marked.js](https://marked.js.org) — Markdown rendering
- Press Start 2P + VT323 — pixel / terminal fonts
- GitHub Pages — hosting

## Aesthetic

BossKey pixel art icons, graph paper background, hard pixel offset box-shadows, warm parchment light theme (`#f0eee8`).
