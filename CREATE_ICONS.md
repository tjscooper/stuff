# Creating App Icons for GAINZ QUEST

You need to create two icon files for the PWA to work properly:

## Required Icons:
- `icon-192.png` (192x192 pixels)
- `icon-512.png` (512x512 pixels)

## Quick Method - Using an Online Tool:

1. Go to https://www.favicon-generator.org/ or https://realfavicongenerator.net/
2. Upload a logo or image (or use text-based icon generator)
3. Generate icons and download
4. Rename the generated 192x192 and 512x512 files to match our requirements

## Design Suggestions:
- Use a simple emoji like 🎮 or 💪
- Background color: `#0f172a` (dark blue from the app)
- Text/Icon color: `#ffd700` (gold)
- Keep it simple and bold for visibility on home screen

## Alternative - Simple Solid Color Icon:

If you just want to test quickly, you can create solid color squares:

1. Use any image editor (Paint, Preview, Photoshop, etc.)
2. Create a 192x192px image with a solid color (e.g., purple `#6366f1`)
3. Save as `icon-192.png`
4. Create a 512x512px version and save as `icon-512.png`

## Using Canva (Easy Option):

1. Go to canva.com
2. Create a custom size design (192x192 or 512x512)
3. Add text "GQ" or an emoji 🎮
4. Add background color: `#0f172a`
5. Add text/icon in gold: `#ffd700`
6. Download as PNG
7. Rename to `icon-192.png` or `icon-512.png`

Place both icon files in the `/stuff/` directory.
